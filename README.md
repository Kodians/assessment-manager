## Run Locally

Clone the project

```bash
  git clone [Project url]
```

Go to the project directory

```bash
  cd [Project Name]
```

Install backend dependencies and build the front-end

```bash
  yarn build
```

Start the back-end

```bash
  yarn back:start
```

Start the front-end

```bash
  yarn front:start
```
